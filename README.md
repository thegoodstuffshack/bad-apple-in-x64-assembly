# bad-apple-in-x64-assembly

## Make

- Download frames from <https://archive.org/details/bad_apple_is.7z>
- Move image_sequence folder to the project directory, or change the 

``` make all ```

## Running in Qemu

``` make run ```

- might need to change mkdir commands

## Running off physical drive

- For USB, copy the contents of the HDA_DRIVE folder into a formatted USB and boot with secure boot disabled.
